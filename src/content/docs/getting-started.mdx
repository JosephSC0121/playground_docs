**_PlaygroundJDC_** Es una aplicación web y de escritorio para aprender a programar. 

<Callout variant="info">
PlaygroundJDC es una aplicación construida con tecnologías como Python, Javascipt, Electron y Go.
</Callout>

## Setup Local
Después de crear tu repositorio local, sigue estos pasos para construir tu propia instancia del proyecto:

Clona el repositorio del backend:

```bash
git clone https://github.com/JosephSC0121/Playground-api
```
Para configurar tu backend en windows:

```bash
cd Playground-api
copy ".env-dev" ".env"
```

Para configurar tu backend en Linux:

```bash
cd Playground-api
cp .env-dev .env
```

<Callout variant="warning">
Recuerda que en el .env tienes que poner tus credenciales de la base de datos, token OPENAI y auth keys. 
</Callout>

una vez configurado el .env:

```bash
docker compose up
```
Frontend:
```bash
git clone https://github.com/JosephSC0121/Playgroud-app
cd Playground-app
npm install
npm run dev
```
Code Executor:
```bash
git clone https://github.com/JosephSC0121/Playground_code_executor
cd Playground_code_executor
go run main.go run standalone
```



Tu backend estará disponible en `http://localhost:80`.

## Revisa la documentación de la api
Para estar al tanto de los endpoints puedes revisar la documentación en `http://localhost:80/docs` 

## Aplicación de escritorio
Al ejecutar los comandos del frontend se ejecutará tu app de electron y en tu navegador en `http://localhost:5173`
